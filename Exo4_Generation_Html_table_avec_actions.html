Exercice 4

Ecrire une page sur le modèle de l’exercice 3 en y ajoutant des boutons ACTIONS (Modifier et Supprimer) comme ceci

Entete1 Entete2 Entete3 Actions
Val11 Val12 Val13
Val21 Val22 Val23
Val31 Val32 Val33
Val41 Val42 Val43



Remarque :
• Chaque tous les boutons MODIFIER auront le même attribut CLASS qui servira à associer l’évènement CLICK.
• Chaque tous les boutons SUPPRIMER auront le même attribut CLASS qui servira à associer l’évènement CLICK (cette CLASS devra être différente de celle du bouton MODIFIER).
• Chaque bouton Modifier et Supprimer aura un attribut Value qui sera égal au N° de la ligne. Lorsque l’on cliquera sur un bouton modifier on devra voir apparaitre un message du type




• Si l’on a cliqué sur le bouton Modifier de la 2eme ligne.
• Ce mecanisme doit fonctionner pour tous les boutons Modifier et Supprimer.
